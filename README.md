# The Water We Eat

This project contains source code for the waterweeat.com website.

## Build

```sh
npm install
```

## Build

```sh
# for production:
npm build

# for development:
npm devbuild
```

## Run locally

```sh
npm run webpack-server
open http://localhost:8080/
```

## Deploy

Copy dist folder contents to remote server
